# Shawnderland OKDO

Unified hub for AI creative tools. One interface, five integrated toolsets.

## Tools

| Tool | Purpose | Stack |
|------|---------|-------|
| AI Sprite Lab | Sprites, pixel art, animations, tilesets, SFX | Next.js, Gemini, FFmpeg |
| ShawnderMind | 8-stage AI ideation pipeline | React, @xyflow/react, Gemini |
| AI UI Lab | Game UI generation, layout planning | PySide6, FastAPI, Gemini |
| AI ConceptLab | Image gen, multiview, character/weapon design | React, @xyflow/react, Imagen/Gemini |
| Walter Storyboarding | Arc-guided video storyboarding | Electron, React, Zustand |

## Status

Onboarding complete. Hub scaffold running. Awaiting feature direction.
