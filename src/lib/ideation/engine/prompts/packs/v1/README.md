# Prompt Packs v1 — Reference Files

These `.md` files are **human-readable references** for the v1 prompt templates.

The runtime source of truth for all prompt templates is the inline `PACK_TEMPLATES['v1']` object in `src/engine/prompts/loadPack.ts`. These reference files are not loaded at runtime.

When editing prompts, update `loadPack.ts` first, then sync these reference files to match.
